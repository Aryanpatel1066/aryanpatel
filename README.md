## 🚀 How to Use
To run the portfolio locally:
1. Clone the repository:
    ```bash
    git clone https://github.com/Aryanpatel1066/aryanpatel
    ```
2. Navigate to the project directory:
    ```bash
    cd aryanpatel
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Run the development server:
    ```bash
    npm start
    ```
